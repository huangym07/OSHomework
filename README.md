## 结构说明：
include/										头文件目录
exp1												编程接口实验

exp3												动态页式存储管理实验

README.md										read me

## 运行环境
Architecture:						x86\_64
Operating System:				Ubuntu 22.04.5 LTS
gcc:										gcc version 11.4.0 (Ubuntu 11.4.0-1ubuntu1~22.04)
make:										GNU Make 4.3 Built for x86\_64-pc-linux-gnu


## 构建方法：
分别进入 exp1 ~ exp4 目录，根据其中的 README.md 进行 make 即可
